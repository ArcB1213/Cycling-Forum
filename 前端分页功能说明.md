# 前端分页功能实现说明

## 📋 实现概述

已成功为前端实现分页功能,包括车手列表页面和赛段成绩页面。

## 🎯 实现的功能

### 1. **车手列表分页** (RidersView.vue)

- ✅ 每页显示 **16 位车手**
- ✅ 页码导航(前后各显示 2 页)
- ✅ 上一页/下一页按钮
- ✅ 显示总车手数和当前页信息
- ✅ 页面内搜索功能保留

### 2. **赛段成绩分页** (RacesView.vue)

- ✅ 每页显示 **20 条成绩记录**
- ✅ 页码导航(前后各显示 2 页)
- ✅ 上一页/下一页按钮
- ✅ 显示总成绩数和当前页信息
- ✅ 切换页面时自动滚动到顶部

## 📝 更新的文件

### 1. 类型定义 (`types.ts`)

```typescript
// 新增分页元数据接口
interface PaginationMeta {
  total: number;
  page: number;
  limit: number;
  total_pages: number;
  has_next: boolean;
  has_prev: boolean;
}

// 新增分页响应接口
interface PaginatedRidersResponse {
  data: Rider[];
  pagination: PaginationMeta;
}

interface PaginatedStageResultsResponse {
  stage_info: Stage;
  data: StageResult[];
  pagination: PaginationMeta;
}
```

### 2. API 服务 (`ApiServices.ts`)

```typescript
// 更新为支持分页参数
export const fetchRiders = async (page: number = 1, limit: number = 16)
export const fetchResults = async (stageId: number, page: number = 1, limit: number = 20)
```
